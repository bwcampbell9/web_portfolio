Project: This Site!

I made this site using HTML and CSS (with Sass) following tutorials from Brad Traversy


In the third quarter of my freshman year at Cal Poly I decided to begin to learn web development. This interest led me of course to HTML and CSS. I always like to learn new things by diving into projects that interest me and this learning experience was no different. Searching online for HTML and CSS tutorials I stumbled on the excellent tutorials of Brad Traversy at the youtube channel Traversy Media.

The tutorial on creating a responsive web portfolio sparked my interest and I realised that it would be the perfect project to start learning web development. After finishing the tutorials I have since continued to expand my knowlage of web development by adding more features to the site like these project pages.

In the future I plan to continue the site as I learn more about web development and design. This website is a way to showcase my skills not only through the information listed on it but also through the website itself.
