Project: White Elephant

A group app development project for a software development class.


This Android application was made for a software development class at cal poly. White Elephant is a tinder-style bartering platform where users swipe on items they would like to trade.

First, we learned about software design and came up with the concept for White Elephant. Next, we used Android Studio to write the application in Java, in addition, we made use of Google Firebase for authentication and data storage.
